## Tech Stack

**Client:** React, React Router, Axios

**Server:** Django, Django REST framework, SQLite

**Tools:** Vite, Jest

## Installation (Backend)


```bash
  cd backend
  pip install -r .\requirements.txt
```

To run server

```bash
  python manage.py runserver
```

## Installation (Frontend)

```bash
  cd frontend
  npm install
```

To run development server

```bash
  npm run dev
```
